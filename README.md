# Grove

Personal finance tracker for Android focused on one number: safe to spend today.

## Run

```sh
./gradlew installDebug
./gradlew testDebugUnitTest
```

## Shape

```text
app/src/main/java/com/grove/app
├── core/format
├── data/model
├── data/db
├── data/repository
├── designsystem
└── feature
```

Java 17, Android SDK 36, Kotlin 2.0.21, Compose, Room, DataStore.
